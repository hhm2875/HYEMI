## ⭐ 주요 기능
* 센서 데이터 수집 : Raspberry Pi를 활용하여 MQ-2, DS18B20, 불꽃감지 센서 데이터를 수집한다.
* 화재 위험 판단 :  수집한 데이터를 db로 전송 후 ai 판단 로직을 적용하여 정상, 주의, 위험 상태 구분
* DB저장: MySQL DB에 저장된 센서 데이터를 구분하고, 그 결과
